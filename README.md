# 🚀 QuickAI – AI-Powered SaaS Platform

QuickAI is a full-stack AI SaaS application built using the PERN stack and powered by local AI models. The platform provides intelligent tools for article generation, blog title generation, resume analysis, and AI image generation through seamless integration of **Llama 3 (via Ollama)** and **Stable Diffusion**.

🌐 **Live Demo:** https://quickai-gs.vercel.app/

---

## ✨ Features

* 📝 **Article Generator**

  * Generate high-quality articles using Llama 3.

* 💡 **Blog Title Generator**

  * Create engaging and SEO-friendly blog titles.

* 📄 **AI Resume Analyzer**

  * Analyze resumes and provide insights for improvement.

* 🎨 **AI Image Generator**

  * Generate images from text prompts using Stable Diffusion.

* 🔒 **Authentication & User Management**

  * Secure user authentication and session management.

* ⚡ **Responsive UI**

  * Optimized user experience across devices.

* ☁️ **Cloud Deployment**

  * Deployed on Vercel for fast and reliable access.
